# Order System

A simple, interactive web-based order system for a restaurant. Users can select starters, main dishes, and desserts, specify quantities, and generate an order bill.

## Features

- **Interactive Slider**: Navigate through different course categories (Starter, Dish, Dessert).
- **Dynamic Order Table**: Add items to your order and view them in a table.
- **Total Calculation**: Automatically calculate the total cost of the order.
- **Responsive Design**: Works on various screen sizes.

## How to Run Locally

1. Clone the repository.
2. Open `index.html` in your web browser.

## GitHub Pages

This project is ready for GitHub Pages.

1. Go to the **Settings** tab of this repository.
2. Click on **Pages** in the left sidebar.
3. Under **Source**, select `Deploy from a branch`.
4. Select the branch (e.g., `main` or `develop`) and folder (`/` root).
5. Click **Save**.

The website will be available at: `https://<your-username>.github.io/<repository-name>/`
